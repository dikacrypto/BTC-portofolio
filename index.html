💾 Backup & Restore Data
Data disimpan di localStorage browser. Agar tidak hilang saat cache dihapus atau reinstall PWA, rutin export dan simpan file JSON-nya.
📌 Tips: Export setiap kali selesai input data harian/bulanan. Simpan file di Google Drive / WhatsApp ke diri sendiri agar aman.
☁️ GitHub Gist — Auto Cloud Sync
GitHub Personal Access Token (scope: gist)
Gist ID (kosongkan → akan dibuat otomatis)
📌 Cara setup (sekali saja):
1. Buka github.com → Settings → Developer settings → Personal access tokens → Tokens (classic)
2. Generate new token → centang scope gist → copy token
3. Paste token di atas → klik Simpan Konfigurasi
4. Klik Upload ke Gist → Gist ID akan terisi otomatis
5. Selesai! Data akan auto-sync setiap kali input data baru ☁️
